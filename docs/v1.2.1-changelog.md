## 修复 1：漏填定位优化

### 问题描述
在 v1.2 中，点击「完成填写并导出」后，如果有必填项未填：
- 只显示 Toast 提示「还有 X 个必填项未完成」
- 仅滚动到第一个未完成的模块，而非具体题目
- 用户需要自己寻找哪个题目未填
- 如果未填的是「其他」补充输入框，无法直接定位到输入框

### 修复方案

**1. 题目级定位**
- 校验时记录第一个未填题目的 ID（`firstMissingQuestionId`）
- 如果是「其他」补充输入框未填，记录 `firstMissingOtherId`
- 优先处理 `other` 未填情况

**2. 错误高亮样式**
新增 CSS 样式：
```css
.question.error {
  border-color: #ff4444;
  background: #fff5f5;
  box-shadow: 0 0 0 2px rgba(255, 68, 68, 0.2);
}

.question.error .question-text {
  color: #ff4444;
}

.other-input-container.error {
  border-color: #ff4444;
  background: #fff5f5;
  box-shadow: 0 0 0 2px rgba(255, 68, 68, 0.2);
}
```

**3. 自动滚动与聚焦**
- 找到题目元素后添加 `error` 类
- 滚动到题目（`scrollIntoView({ block: 'center' })`）
- 聚焦到第一个输入控件
- 如果是 `other` 输入框，先展开再滚动再聚焦

**4. 错误状态自动清除**
- 用户开始填写时（`saveAnswer`/`saveAnswerCheckbox`/`saveAnswerWithDebounce`）
- 自动清除对应题目的 `error` 类
- 给用户即时反馈

### 代码变更

**validateAndComplete 函数重构**：
- 新增 `firstMissingQuestionId` 和 `firstMissingOtherId` 追踪
- 优先处理 `other` 补充输入框未填
- 展开输入框 → 添加 error 类 → 滚动 → 聚焦

**saveAnswer 函数增强**：
- 用户填写时清除题目 error 状态
- 用户填写时清除 other 输入框 error 状态

**saveAnswerCheckbox 函数增强**：
- 用户填写时清除题目 error 状态
- 用户填写时清除 other 输入框 error 状态

**saveAnswerWithDebounce 函数增强**：
- 文本输入时清除题目 error 状态

## 修复 2：顶部防闪动优化

### 问题描述
在 v1.2 中，顶部 header 在滚动时存在闪动问题：
- 向下滚动时收起，向上滚动时展开
- 但在临界位置（约 80px）附近轻微上下滚动时
- header 会频繁切换 collapsed 状态
- 导致视觉上「抖动」「闪动」「不自然」

### 修复方案

**1. 滞回区间（Hysteresis）**
- 收起阈值：100px（向下滚动超过 100px 才收起）
- 展开阈值：50px（向上滚动超过 50px 才展开）
- 在 50px-100px 区间内保持当前状态不变

**2. 最小滚动距离判断**
- `minScrollDelta = 10px`
- 滚动距离小于 10px 视为轻微抖动，不触发状态切换

**3. 滚动方向判断**
- 明显向下（`scrollDelta > 10px`）才考虑收起
- 明显向上（`scrollDelta < -10px`）才考虑展开

**4. 状态保持**
- 使用 `lastScrollY` 追踪上次滚动位置
- 避免在临界位置反复 add/remove class

### 代码变更

**setupHeaderScroll 函数重构**：
```javascript
function setupHeaderScroll() {
  const header = document.querySelector('.header');
  let lastScrollY = window.scrollY;
  let scrollDelta = 0;
  const collapseThreshold = 100;  // 收起阈值
  const expandThreshold = 50;     // 展开阈值（滞回）
  const minScrollDelta = 10;      // 最小滚动距离

  window.addEventListener('scroll', () => {
    const currentScrollY = window.scrollY;
    scrollDelta = currentScrollY - lastScrollY;
    
    const isScrollingDown = scrollDelta > minScrollDelta;
    const isScrollingUp = scrollDelta < -minScrollDelta;
    
    if (isScrollingDown && currentScrollY > collapseThreshold) {
      header.classList.add('collapsed');
    }
    else if (isScrollingUp && currentScrollY < expandThreshold) {
      header.classList.remove('collapsed');
    }
    
    lastScrollY = currentScrollY;
  }, { passive: true });
}
```
