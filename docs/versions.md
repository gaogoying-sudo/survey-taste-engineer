## v1.1（能力基线）

**部署时间**：2026-03-11 12:35

**状态**：能力基线，保留不动

**核心变更**（相比 v1.0）：

### 新增功能
1. **自动保存机制**
   - radio/checkbox/likert 每次操作后立即保存
   - text/textarea 输入停止后防抖 500ms 保存
   - 顶部和底部状态区长期显示「已自动保存 HH:mm:ss」

2. **底部动作区优化**
   - 「保存进度」按钮降级为「立即保存备份」辅助按钮
   - 「提交」按钮改名为「完成填写并导出」
   - 点击后校验通过直接导出 JSON

3. **自动保存状态展示**
   - 长期可见，不完全消失
   - 保存中显示橙色，已保存显示绿色

**文件**：`index-survey-collect-v1.1.html`（64 KB）

**访问链接**：
- 线上：https://survey-taste-engineer.vercel.app/v1.1
- 本地：`file:///path/to/index-survey-collect-v1.1.html`

**备注**：v1.1 作为自动保存与底部动作区能力基线，后续版本均继承这些能力。

## v1.2.1（交互修复）

**部署时间**：2026-03-11 21:58

**状态**：最新修复版本，独立测试链接

**核心变更**（相比 v1.2）：

### 问题 1：漏填定位修复
- 点击「完成填写并导出」后，如有必填项未填，自动滚动到第一个未填的具体题目
- 题目增加红色错误高亮样式（边框变红、背景变浅红、外发光）
- 如果是「其他」补充输入框未填，先展开输入框再滚动定位
- 用户开始填写后，错误高亮自动消失

### 问题 2：顶部防闪动修复
- 增加滞回区间：收起阈值 100px，展开阈值 50px
- 增加最小滚动距离判断（10px），轻微抖动不触发
- 增加滚动方向判断：明显向下才收起，明显向上才展开
